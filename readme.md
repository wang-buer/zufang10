# 基于SSM的房屋出租管理系统10



## 1、项目介绍

基于SSM的房屋出租管理系统拥有两种角色

管理员：基础数据管理、公告管理、出租管理、小区管理、卖房管理等

用户：登录注册、查看各种信息、发布出租信息的

## 2、项目技术

后端框架：SSM（Spring、SpringMVC、Mybatis）

前端技术：前端html、后端VUE前后分离

## 3、开发环境

- JAVA版本：JDK1.8（最佳）
- IDE类型：IDEA、Eclipse都可运行
- 数据库类型：MySql（5.7、8.x版本都可） 
- tomcat版本：Tomcat 7-10版本均可
- maven版本：无限制
- 硬件环境：Windows
